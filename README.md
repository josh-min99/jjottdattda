# 쪼뜯따 (JJOTTDATTDA)

**2인 온라인 전략 대전 게임** - 바이러스를 퍼뜨려 영토를 장악하라!

---

## 게임 소개

쪼뜯따는 두 명의 플레이어가 각각 바이러스를 퍼뜨리며 지도 위의 영토를 차지하기 위해 경쟁하는 **실시간 전략 게임**입니다.

육각형 타일로 이루어진 지도에서 상대보다 더 많은 영토를 감염시키고, 다양한 바이러스 전략과 아이템을 활용해 승리를 쟁취하세요!

---

## 게임 방식

### 기본 규칙
- **2명**의 플레이어가 **레드팀** vs **블루팀**으로 대결
- 총 **6라운드**, 라운드당 **3턴** (총 18턴)
- 각 턴마다 타일을 클릭해 바이러스를 퍼뜨릴 수 있음
- 6라운드가 끝났을 때 더 많은 영토를 차지한 플레이어가 승리!

### 핵심 플레이
1. **바이러스 선택** - 5가지 바이러스 중 전략에 맞는 것을 선택
2. **타일 공격** - 지도의 타일을 클릭해 감염 시도
3. **아이템 활용** - 상점에서 아이템을 구매하고 전략적으로 사용
4. **백신 방어** - 공격뿐 아니라 방어 모드로 내 영토를 보호

---

## 바이러스 종류

| 바이러스 | 특징 |
|---------|------|
| **기본 바이러스** | 균형 잡힌 기본형 |
| **러쉬 바이러스** | 빠르고 공격적인 확산 |
| **앰부시 바이러스** | 은신 - 상대에게 중립으로 보임 |
| **갬블 바이러스** | 고위험 고수익 도박형 |
| **뮤턴트 바이러스** | 특수 버프 효과 부여 |

※ 특수 바이러스는 게임당 사용 횟수 제한이 있습니다.

---

## 주요 시스템

### 스탯 시스템
- **감염률** - 타일 감염 성공 확률
- **치사율** - 감염 시 인구 사망 비율
- **전파력** - 주변 지역으로 퍼지는 힘
- **백신 회복률/공급률** - 방어 능력

### 경제 시스템
- 감염 성공, 영토 확보 시 **골드** 획득
- 상점에서 **스탯 업그레이드** 및 **아이템** 구매 가능

### 특수 아이템
- **폭탄** - 지역 초토화
- **생화학 무기** - 치사율 대폭 증가
- **유전자 가위** - 유전자 조작
- **정부 문서** - 비밀 정보 획득
- **드론** - 특정 지역 면역 부여

---

## 게임 특징

- **실시간 온라인 대전** - 친구와 4자리 방 코드로 간편하게 매칭
- **전략적 깊이** - 바이러스 선택, 타이밍, 자원 관리의 조합
- **확률 기반 전투** - 감염률에 따른 긴장감 있는 결과
- **다양한 이벤트** - 라운드 중 발생하는 랜덤 이벤트로 전략 변화

---

## 플레이 방법

1. 게임 실행
2. 방 만들기 또는 방 코드로 참가
3. 상대 플레이어가 입장하면 게임 시작
4. 매 턴 바이러스를 선택하고 타일을 클릭해 공격
5. 6라운드 후 더 많은 영토를 가진 플레이어 승리!
